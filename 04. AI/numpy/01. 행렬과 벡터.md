# 행렬 표현
2차원 좌표계에서 (3,4)는 x = 3 , y  = 4인 좌표 하나를 표현할 수 있음
벡터는 이에 더해 해당 방향으로의 크기를 표현 가능
## 벡터
좌표 A(3, 4)에 대해, 벡터는 기준점(원점)으로부터 해당 좌표까지의 방향과 거리를 가짐
기준이 (0, 0)일 때 좌표 A에 대하여
- 벡터의 방향 : (3, 4)
- 벡터의 크기 : 5 (피타고라스 정리에 의함)
$$v = \begin{pmatrix}
3 \\
4
\end{pmatrix}$$
다음과 같이 행렬을 표현 가능
numpy에서는 다음과 같이 표현한다
```python
import numpy as np

v = np.array([3, 4])
print(v)
print(np.linalg.norm(v))
```

AI에서의 기본 입력값은 모두 numpy를 사용하므로 익숙해질 필요 있음
