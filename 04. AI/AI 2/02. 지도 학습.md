# 1. 지도 학습

## 1-1. 지도 학습이란?
- **정답 라벨이 있는 훈련 데이터**를 사용해 예측 모델을 학습하는 방법
- 기존 데이터를 바탕으로 **처음 보는 데이터에서도 결과를 잘 예측하는 모델** 만들기
- 목표 : 훈련 데이터 뿐 아니라 처음 보는 데이터에서도 **예측 성능 향상**
## 1-2. 지도 학습의 종류

### 회귀
예측하고자 하는 값이 숫자
### 분류
예측하고자 하는 값이 범주 (O/X, 개/고양이)
## 1-3. 지도 학습 관련 용어

### 특성(피쳐, x)
- 예측에 사용되는 설명 변수
- 예측 함수 $f$의 input
### 라벨(y)
- 맞춰야 하는 정답
### 예측값 $\hat{y}$
- 학습 과정에서 모델이 내놓은 결과(숫자든 범주든)
### 오류

$\hat{y} - y$ (예측값과 라벨(정답)의 차이)
오류 ≠ 측정 오차

# 2. 회귀
- 여러 피쳐로부터 얼마나 정확한 수치를 예측해 내는가?
## 회귀 오류 : 평균제곱오차(MSE)
예측이 얼마나 정확한지 어떻게 아는가? → 알 수 있는 지표 필요
$$MSE = \frac{1}{n}\sum_{i=1}^{n}(y_{i} - \tilde{y_i})^2$$
각 데이터에서 정답과 예측의 오차의 제곱의 평균
틀린 정도를 제곱 → 오류가 커질수록 더 큰 패널티
