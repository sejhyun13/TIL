# 1. 지도 학습

## 1-1. 지도 학습이란?
- **정답 라벨이 있는 훈련 데이터**를 사용해 예측 모델을 학습하는 방법
- 기존 데이터를 바탕으로 **처음 보는 데이터에서도 결과를 잘 예측하는 모델** 만들기
- 목표 : 훈련 데이터 뿐 아니라 처음 보는 데이터에서도 **예측 성능 향상**
## 1-2. 지도 학습의 종류

### 회귀
예측하고자 하는 값이 숫자
### 분류
예측하고자 하는 값이 범주 (O/X, 개/고양이)
## 1-3. 지도 학습 관련 용어

### 특성(피쳐, x)
- 예측에 사용되는 설명 변수
- 예측 함수 $f$의 input
### 라벨(y)
- 맞춰야 하는 정답
### 예측값 $\hat{y}$
- 학습 과정에서 모델이 내놓은 결과(숫자든 범주든)
### 오류

$\hat{y} - y$ (예측값과 라벨(정답)의 차이)
오류 ≠ 측정 오차

# 2. 회귀
- 여러 피쳐로부터 얼마나 정확한 수치를 예측해 내는가?
## 회귀 오류 : 평균제곱오차(MSE)
예측이 얼마나 정확한지 어떻게 아는가? → 알 수 있는 지표 필요
$$MSE = \frac{1}{n}\sum_{i=1}^{n}(y_{i} - \tilde{y_i})^2$$
각 데이터에서 정답과 예측의 오차의 제곱의 평균
틀린 정도를 제곱 → 오류가 커질수록 더 큰 패널티 부여
MSE를 root한 RMSE도 사용함

## 회귀 설명력 : $R^2$

![](attachments/Pasted%20image%2020260318003529.png)

라벨의 분산 중 피쳐로 설명되는 비율
1이 높을수록 설명력이 높음
- 음수일 수 있음 → 예측값들이 평균만도 못한 경우
## MSE vs $R^2$
모델 자체에 대한 평가는 $R^2$를 통해 표현하는 것이 좋음(1에 가까울수록 더 설명력이 높음이 직관적으로 드러남)
MSE는 실제로 어느 정도의 오차가 있는지 비교하기 용이
# 3. 분류
- 각 피쳐로부터의 라벨이 범주로 나타남
- O/X or A인가? B인가? C인가?
- 특정 선을 기준으로 데이터를 구분
- 분류 모델은 라벨이 ‘범주’로 나타나기 때문에 MSE나 $R^2$를 쓰기 어려움
## 분류 정확도
![](attachments/Pasted%20image%2020260318004049.png)
(혼동 행렬(오분류표), Confusion Matrix)
$$\text{Accuracy} = \frac{\text{TP} + \text{TN}}{\text{TP} + \text{TN} + \text{FP} + \text{FN}}$$
즉 **올바르게 맞춘 개수 / 전체 예측 개수**
