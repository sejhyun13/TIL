# 재귀 함수와 내부 `return`의 역할
2026-02-20  
return은 함수를 종료하는게 아니라, 그 이전 단계로 돌아가게끔 한다

## 재귀함수에서의 사용
f(n) 내부에 f(n+1)을 호출하는 식의 재귀함수를 예시로 볼 때 작업은 다음과 같이 이루어진다.  
1. f(1) 실행시 스택 메모리에 f(1) 추가
2. f(2) 실행, 스택 메모리의 f(1)위에 f(2) 추가, f(1)은 f(2) 호출 종료시까지 일시정지
3. f(3) 실행, 이하 반복

특정 종료 조건을 만족하여 f(i)의 return문이 실행될 경우, 프로세스는 쌓인 스택 메모리에서 하나씩 제거하는 방식으로 이전 단계로 돌아간다.  
f(5)에서 조건을 만족하여 `return`이 실행될 때
1. f(5) 종료, 일시정지된 f(4)로 돌아감
2. f(4) 종료, 이하 반복

## 알아야 하는 것
return의 기능은 함수 종료 또는 값 반환의 역할만 하지 않는다. 이전 단계의 상황으로 돌아가게끔 하며, 10번 중첩된 재귀함수일 경우 return문이 10번 실행되어야 맨 처음 상태로 돌아갈 수 있다.
