## 1. 개념 및 목적

이차원 리스트(Matrix) 구조에서 특정 정점 $(y, x)$로부터 인접한 정점으로 이동하기 위한 **상대적 변위 차(Delta)**를 관리하는 기법입니다. 수동으로 `if`문을 나열하는 방식의 비효율성을 제거하고, 반복 가능한 구조로 로직을 정형화하는 데 목적이 있습니다.

## 2. 상세 작동 원리

### 2.1. 델타 배열(Delta Array)의 구조

격자에서 한 칸 이동한다는 것은 좌표값에 정수 연산을 가하는 것입니다.

- **상(Up):** 행 인덱스가 1 감소 $\rightarrow (y-1, x)$
    
- **하(Down):** 행 인덱스가 1 증가 $\rightarrow (y+1, x)$
    
- **좌(Left):** 열 인덱스가 1 감소 $\rightarrow (y, x-1)$
    
- **우(Right):** 열 인덱스가 1 증가 $\rightarrow (y, x+1)$
    

이 변화량(Delta)을 각각 `dy = [-1, 1, 0, 0]`와 `dx = [0, 0, -1, 1]`라는 독립된 배열로 선언합니다. 이때 두 배열의 **동일한 인덱스**는 하나의 방향 쌍을 이룹니다. (예: `dy[0]`와 `dx[0]`를 합치면 '상' 방향 벡터가 됨)

### 2.2. 벡터 합성 및 전이(Transition)

현재 위치를 $P(y, x)$라고 할 때, 다음 위치 $P'(ny, nx)$는 다음과 같은 벡터 합성으로 결정됩니다.

$$ny = y + dy[i]$$

$$nx = x + dx[i]$$

여기서 $i$는 방향 인덱스이며, 0부터 (방향 개수 - 1)까지 순회하며 인접한 모든 칸을 조사합니다.

### 2.3. 경계 검사(Boundary Check)의 필수성

컴퓨터의 메모리 구조상 배열의 인덱스는 $0$부터 $N-1$까지만 유효합니다. 계산된 $ny, nx$가 이 범위를 벗어날 경우 `IndexError`가 발생하므로, 실제 데이터에 접근하기 전 반드시 다음 조건을 검증해야 합니다.

1. $0 \le ny < \text{Row Size}$
    
2. $0 \le nx < \text{Col Size}$
    

## 3. 예시 코드

```python
# 4방향 탐색 (상, 하, 좌, 우)
dy = [-1, 1, 0, 0]
dx = [0, 0, -1, 1]

def navigate(y, x, N):
    print(f"현재 위치: ({y}, {x})")
    for i in range(4):
        ny = y + dy[i]
        nx = x + dx[i]
        
        # 격자 범위(N x N) 내에 있는지 확인
        if 0 <= ny < N and 0 <= nx < N:
            print(f"방향 {i} 이동 성공: ({ny}, {nx})")
        else:
            print(f"방향 {i} 이동 실패: 범위를 벗어남")

# 실행 예시 (3x3 격자의 중앙 (1, 1)에서 탐색)
navigate(1, 1, 3)

"""
예상 출력:
현재 위치: (1, 1)
방향 0 이동 성공: (0, 1)
방향 1 이동 성공: (2, 1)
방향 2 이동 성공: (1, 0)
방향 3 이동 성공: (1, 2)
"""

# 실행 예시 (3x3 격자의 모서리 (0, 0)에서 탐색)
# navigate(0, 0, 3)
"""
예상 출력:
현재 위치: (0, 0)
방향 0 이동 실패: 범위를 벗어남
방향 1 이동 성공: (1, 0)
방향 2 이동 실패: 범위를 벗어남
방향 3 이동 성공: (0, 1)
"""
```

## 4. 기술적 분석 (객관적 평가)

- **효율성:** `if`문을 개별적으로 4번 쓰는 것보다 코드의 가독성이 높고, 방향이 늘어나도 로직의 복잡도가 일정하게 유지됩니다.
    
- **확장성:** 8방향 탐색이나 체스의 '나이트' 이동처럼 복잡한 이동 패턴도 델타 배열만 수정하여 쉽게 구현 가능합니다.
    
- **유의점:** 파이썬 리스트의 `matrix[y][x]` 접근 방식에서 `y`가 세로축(행), `x`가 가로축(열)임을 인지하고 `dy`, `dx`의 순서를 일치시켜야 합니다.
