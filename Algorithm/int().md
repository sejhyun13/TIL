# Python: int() 함수의 기능과 활용

* **날짜:** 2026-03-09
* **카테고리:** #Python #Programming_Basics #TypeConversion

## 1. 개념 및 목적

`int()` 함수는 전달받은 숫자나 문자열을 **정수(Integer)** 형태로 변환하여 반환하는 파이썬 내장 함수입니다. 실수에서 소수점을 제거하거나, 다양한 진법의 문자열을 10진수 정수로 계산할 때 사용합니다.

## 2. 상세 작동 원리

### 2.1. 인자 구성

`int(x, base=10)` 형태로 사용되며, 두 개의 인자를 가질 수 있습니다.

* **`x`:** 변환할 숫자 또는 문자열입니다. 인자를 넣지 않으면 `0`을 반환합니다.
* **`base` (선택 사항):** `x`가 문자열일 때, 해당 문자열이 어떤 진법인지 명시합니다. 기본값은 10이며, 2부터 36까지 설정 가능합니다.

### 2.2. 주요 변환 규칙

1. **실수 -> 정수:** 소수점 이하를 내림이나 반올림하지 않고 **무조건 버림(Truncation)** 처리합니다. (예: `int(3.9)` -> `3`)
2. **문자열 -> 정수:** 문자열 내의 숫자를 해석합니다. 앞뒤의 공백은 허용되지만, 숫자 이외의 문자가 포함되면 `ValueError`가 발생합니다.
3. **불리언 -> 정수:** `True`는 `1`, `False`는 `0`으로 변환됩니다.

## 3. 예시 코드

다양한 상황에서 `int()` 함수가 어떻게 작동하는지 보여주는 코드입니다.

```python
# 1. 기본적인 형변환
print(int(3.14))    # 3 (소수점 버림)
print(int("-10"))   # -10 (음수 문자열 처리)
print(int(True))    # 1 (Boolean 처리)

# 2. 진법 변환 (base 인자 활용)
print(int('101', 2))   # 5 (2진수 101을 10진수로)
print(int('A', 16))    # 10 (16진수 A를 10진수로)
print(int('12', 8))    # 10 (8진수 12를 10진수로)

# 3. 에러 케이스
try:
    int("3.14") # 문자열 내에 소수점이 있으면 바로 정수 변환 불가
except ValueError as e:
    print(f"오류: {e}")
# 해결법: float으로 먼저 바꾼 후 int로 변환 -> int(float("3.14"))

```

## 4. 기술적 분석
* **메모리 관리:** 파이썬 3의 `int`는 'Arbitrary Precision'을 지원합니다. 즉, 메모리가 허용하는 한 **무한한 크기의 정수**를 다룰 수 있어, 다른 언어에서 발생하는 정수 오버플로(Overflow) 문제에서 자유롭습니다.
* **성능:** `int(string)` 변환은 문자열의 길이에 비례하는 시간 복잡도($O(n)$)를 가집니다. 매우 긴 숫자가 담긴 문자열을 정수로 바꿀 때는 성능 저하가 발생할 수 있습니다.
* **유연성:** `base=0` 설정을 통해 접두사 기반의 자동 진법 판별이 가능하다는 점은 파이썬 `int()`만의 강력한 장점입니다.
