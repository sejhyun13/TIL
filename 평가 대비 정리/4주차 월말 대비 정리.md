# 1. 2진수/16진수/대문자
## 2진수 변환
### 10 → 2진수
65를 2진수로 변환한다고 할 때
1. 65 / 2 = 32 + 1
2. 32 / 2 = 16 + 0
3. 16 / 2 = 8 + 0
4. 8 / 2 = 4 + 0
5. 4 / 2 = 2 + 0
6. 2 / 2 = 1 + 0
7. 1 / 2 = 0 + 1
→ 따라서 65의 2진수는 1000001 이다
→ 계속 2로 나누면서 그 나머지를 붙여 나가면 됨

#### 실수 변환은 어떻게?
0.652 를 2진수로 변환
1. 0.652 x 2 = 1.34 → 1 추가
2. 0.34 x 2 = 0.78 → 0 추가
3. 0.78 x 2 = 1.56 → 1 추가
이런 식으로 소수부가 0이 될 때까지 더한다
### 2 → 10진수
1101 을 10진수로 변환
1. 1 x 2^0 = 1
2. 0 x 2^1 = 0
3. 1 x 2^2 = 4
4. 1 x 2^3 = 8
→ 1+0+4+8 = 13

# 2. 완전 탐색, 부분집합의 합
완탐은 질리도록 했으니 패스
## 부분집합의 합
```python
def generate_subsets_with_bits(arr):
    n = len(arr)
    all_subsets = []
    
    # 1. 0부터 2^n - 1까지 모든 비트 상태 순회
    for i in range(1 << n):
        subset = []
        # 2. i의 j번째 비트가 1인지 확인하여 원소 추출
        for j in range(n):
            if i & (1 << j):
                subset.append(arr[j])
        all_subsets.append(subset)
    
    return all_subsets
```
1 << n = 2^n 
`if i & (1 << j)` → i와 2^j 중 겹치는 비트가 있는 경우 해당 인덱스의 원소 추가
