# 외판원 문제(Traveling Salesman Problem, TSP)

* **날짜:** 2026-03-09

- 재귀와 백트래킹을 활용하여 문제를 해결해야 함
- 경로를 방문하고, 다시 퇴각할 때 방문 처리를 해제하는 과정(백트래킹)이 필요

## 기억해야 하는 것
- 백트래킹은 반드시 끝까지 방문 후 `return`으로 퇴각할 때 기존 방문 처리를 해제해야 한다.
- 
## SWEA 5189.전자카트 풀이(재귀 & 백트래킹 활용)
```python
def visit(curr, count, current_total):
    global min_result
    
    # 가지치기: 이미 현재 최솟값보다 크면 더 볼 필요 없음
    if current_total >= min_result:
        return

    # 모든 지점을 방문했으면 0으로 돌아감
    if count == N - 1:
        min_result = min(min_result, current_total + arr[curr][0])
        # 최솟값은 어차피 전체 중 하나이므로 기존 최솟값과 비교 후 퇴각(더 작으면 최솟값 갱신)
        return

    for next_node in range(1, N):
        if not vis[next_node]:
            vis[next_node] = True # 방문 체크
            visit(next_node, count + 1, current_total + arr[curr][next_node])
            vis[next_node] = False # 핵심: 백트래킹 (방문 해제)

T = int(input())
for tc in range(1, T + 1):
    N = int(input())
    arr = [list(map(int, input().split())) for _ in range(N)]
    
    min_result = float('inf')
    vis = [False] * N
    
    # 0번에서 시작
    visit(0, 0, 0)
    
    print(f"#{tc} {min_result}")
```
